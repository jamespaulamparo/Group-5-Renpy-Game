## ‚öôÔ∏è Installation & Usage

### Prerequisites
To run the game, you must have the **Ren'Py SDK** installed. (a working standalone app is not yet implemented.)

### How to Run:
1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/jamespaulamparo/Group-5-Renpy-Game.git](https://github.com/jamespaulamparo/Group-5-Renpy-Game.git)
    ```
2.  **Open in Ren'Py:** Place the `Group-5-Renpy-Game` folder inside your Ren'Py SDK's `projects` directory.
3.  **Launch:** Open the Ren'Py Launcher, find the "Group 5" project, and click **Launch Project**.
